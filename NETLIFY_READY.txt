🚀 DEPLOYMENT OPTIONEN
----------------------

Option 1: DRAG & DROP (Empfohlen für Einsteiger)
   1. Terminal öffnen im Projekt-Ordner
   2. Ausführen: npm install && npm run build
   3. Öffnen: https://app.netlify.com/drop
   4. Den 'out' Ordner hochziehen → Fertig!

Option 2: GITHUB + NETLIFY
   1. Code auf GitHub pushen
   2. Bei Netlify einloggen
   3. "Import from Git" wählen
   4. Repository verbinden → Automatische Deployment!
